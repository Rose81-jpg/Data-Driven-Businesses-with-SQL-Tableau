Project Reflection: E-Commerce Delivery and Performance AnalysisSummaryThis project features a comprehensive Tableau workbook designed to analyze and visualize e-commerce operational data. The analysis explores the entire order lifecycle—from purchase and approval to final delivery—while also examining customer and seller geographic distributions. Key focus areas include delivery time performance, payment method trends, and product category popularity.Languages and Libraries UsedTableau Desktop (Version 2026.1): Primary tool for data visualization and dashboard creation.SQL: Used for initial data extraction and aggregation before importing into Tableau.CSV/Flat Files: Served as the underlying data storage for various tables including orders, customers, and products.Key LearningsData Integration: Practiced joining multiple disparate datasets (Orders, Payments, Reviews, and Geo-data) into a unified relational model within Tableau.Geospatial Visualization: Gained insights into mapping customer and seller locations using latitude and longitude coordinates.Time-Series Analysis: Analyzed delivery performance by calculating the variance between estimated and actual delivery dates.Operational Communication: Developed the ability to translate complex SQL-extracted metrics into visual dashboards that can be easily understood by non-technical business stakeholders.Challenges OvercomeData Path Management: Resolved issues related to absolute file paths (C:\Users...) when moving the workbook between different local environments by ensuring all data sources were properly extracted.Schema Mapping: Managed complex relationships between diverse tables, such as connecting product IDs across items and category translation files to ensure accurate reporting.Additional ReflectionsThis project highlighted the importance of clean data preparation at the SQL level to simplify the visualization process in Tableau. Future iterations could include predictive modeling to estimate delivery delays before they occur based on seller location and product dimensions.
